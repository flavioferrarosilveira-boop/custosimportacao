# ImportCalc Brasil 🌍

**Simulador profissional de custos de importação para o Brasil**

Acesse: https://seu-usuario.github.io/importcalc-brasil

---

## Funcionalidades

- ✅ Cálculo completo de **Landed Cost** (CIF + Tributos + Despesas)
- ✅ **11 Incoterms® 2020** com campos dinâmicos
- ✅ Tributação completa: **II · IPI · PIS · COFINS · AFRMM · ICMS · DUIMP**
- ✅ **Regimes tributários**: Integral, Redução, Suspensão, Isenção, Não Incidência
- ✅ **Autocomplete NCM** via API Receita Federal (Portal Único SISCOMEX)
- ✅ **Cotações de câmbio** automáticas (ExchangeRate-API)
- ✅ **Frete de Referência** estimado por modal e origem
- ✅ **Seguro Internacional** por valor fixo ou percentual
- ✅ **Mapa de rota** interativo (Leaflet + OpenStreetMap)
- ✅ Painel **BI com gráficos** (Chart.js)
- ✅ Exportação **PDF** e **Excel** (SheetJS)
- ✅ **Manual de preenchimento** integrado

## Metodologia

| Tributo | Base Legal |
|---|---|
| II — Imposto de Importação | Art. 20 CTN · Dec.-Lei 37/1966 |
| IPI | Art. 47 CTN · Dec. 11.158/2022 |
| PIS-Importação | Lei 10.865/2004 art. 7º I |
| COFINS-Importação | Lei 10.865/2004 art. 7º I |
| AFRMM | Lei 10.893/2004 art. 6º |
| ICMS — por dentro | LC 87/1996 art. 13 §1º II "a" |
| Taxa DUIMP/SISCOMEX | Portaria ME 4.131/2021 |

Valoração aduaneira: **Decreto 6.759/2009** (Acordo de Valoração Aduaneira — GATT/OMC)

## Como publicar no GitHub Pages

1. Faça fork ou clone este repositório
2. Vá em **Settings → Pages**
3. Em **Source**: selecione `Deploy from a branch`
4. Em **Branch**: selecione `main` e pasta `/ (root)`
5. Clique em **Save**
6. Aguarde ~1 minuto e acesse a URL gerada
